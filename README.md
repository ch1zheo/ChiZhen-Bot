# ChiZhen Bot

## Информация

ChiZhen Bot - это база для бота-ИИ в Telegram. При желании вы можете изменить название нейросети для индивидуализации.

MIT License - Используйте как хотите, но указывайте автора.

В самом конце будут указаны все строки где применяется название ChiZhen.

ChiZhen Bot реализован на бесплатных API от компании Groq, при желании вы можете изменить ИИ-модель применяемую для ответов пользователю.

ChiZhen Bot имеет высокую защиту от джейлбрейков и создания чего-либо вредоносного, блокируются все темы связанные с оружием, наркотиками и чем-либо преступным.

# Схема работы:

Бот принимает сообщение от пользователя, проверяет его на безопасность, передаёт в Groq API, получает ответ и отправляет обратно. Всё асинхронно, без блокировок.

Пользователь → Telegram → Бот
   -> проверка безопасности (safety.py)
   -> добавление в историю (conversation_manager)
   -> запрос к Groq API (groq_client.py)
   -> сохранение ответа в историю
   -> ответ пользователю

# Установка и запуск

# Windows

1. Установить Python 3.10 или выше
Скачать с python.org, при установке отметить "Add to PATH"

2. Открыть терминал (cmd или PowerShell) в папке с проектом
cd C:\путь_к_папке_бота

3. Создать виртуальное окружение
python -m venv venv

4. Активировать
venv\Scripts\activate

5. Установить зависимости
pip install -r requirements.txt

6. Создать и заполнить `.env` токенами
Cоздайте файл `.env`, скопируйте содержимое `default.env.example` в созданный файл, возьмите свои токены от Groq и @BotFather и заполните в соответствующие поля.

7. Запустить
python main.py

# Где взять токены?

BOT_TOKEN - в Telegram найти бота @BotFather -> /newbot, далее по инструкции. После создания вам выдадут токен.

GROQ_API_KEY - зарегистрироваться в Groq Cloud, перейти по ссылке -> https://console.groq.com/keys, получить API ключ.

По желанию можете изменить модель (GROQ_MODEL) перейдите по ссылке -> https://console.groq.com/settings/limits, там список всех моделей, выбираете нужную и заменяете настоящую на новую.
